New CROC test board - 12 Nov 2019 

-> This board has 2 50-pin connectors to directly connect with BEB3.
-> Also, there is only one SMA input and there are 4 relays to send the signal to each channel input.


Power supply
	+12 V -> ~0.370 A *
	-5  V ->  0.024 A
	+3.3V ->  0.02  A
	*This consumption is much higher from the previous configuration with the CROC_to_BEB board (~0.190A)
